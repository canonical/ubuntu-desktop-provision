## Configuration

Supported formats:
- [YAML](https://yaml.org/) (`.yaml`, `.yml`)

    ```yaml
    bootstrap:
      pages:
        - locale
        - keyboard
        - source
        - storage

    init:
      pages:
        - timezone
        - identity
    ```

Lookup order:
- `/etc/ubuntu-provision.{yaml,yml}` (admin)
- `/usr/local/share/ubuntu-provision.{yaml,yml}` (oem)
- `/usr/share/ubuntu-provision.{yaml,yml}` (distro)
